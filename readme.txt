Workible’s FREE Jobs@ site plugin creates your own Careers page in seconds.

== Description ==

Using Workible’s technology and infrastructure to power your own Custom Careers Page is easy and cost effective.
Workible’s FREE Jobs@ site plugin creates your own Careers page in seconds. Jobs@ blends seamlessly into your existing website while Workible keeps your Careers page up to date and the application process simple. Candidates apply directly from your website – and you’ll still get access to Workible’s candidate ranking, filtering and video interviewing systems.

Take your Career Page to the next level by combining it with your own Workible Job Network to make advertising vacancies and distributing your organisation’s profile to pre-qualified job-searching candidates easy and cost effective.

== Installation ==

1. Upload the workible-widget to the '/wp-content/plugins/' directory, or install the plugin through the WordPress Admin.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings->Workible Widget page to add your affiliate ID.
4. Insert workible widget code to your desired page.
5. [wk-job-list] - Job list shortcode.
6. [wk-job-search] - Job search shortcode
